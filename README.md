# Word Garden

Статическое приложение для GitHub Pages: общая коллекция английских слов и карточки для изучения.

## Подключение Supabase

1. Создайте проект на [Supabase](https://supabase.com/), откройте **SQL Editor** и выполните `supabase/schema.sql`.
2. Откройте сайт, перейдите в «Настройки» и вставьте Project URL и publishable/anon key из Supabase.

## Публикация на GitHub Pages

Загрузите эти файлы в репозиторий GitHub. В **Settings → Pages** выберите публикацию из ветки `main`, папка `/ (root)`.

> Перевод выполняется через MyMemory Translation API без ключа. Бесплатный лимит — 5 000 символов в день.

> Сейчас SQL-политики дают всем посетителям сайта доступ к одной коллекции. Это удобно для личного сайта, но не подходит для публичного сайта: тогда нужно подключить авторизацию Supabase и изменить политики доступа.
