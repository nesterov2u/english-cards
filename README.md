# English Cards

Минималистичное PWA для изучения английских слов и фраз. Карточки хранятся в общей базе Supabase: приложение подставляет перевод и краткое русское объяснение, а затем показывает слова в перемешанной учебной сессии.

## Возможности

- раздел «Перевод» для добавления карточки: слово или фраза, перевод и объяснение;
- автозаполнение через MyMemory и Dictionary API с возможностью исправить данные перед сохранением;
- раздел «Словарь» с постраничной подгрузкой карточек, редактированием и удалением;
- раздел «Учить» с перемешанной колодой без повторов в рамках сессии, прогрессом и финальной карточкой;
- управление учебной карточкой касанием, `Enter` или `Space`;
- адаптивный интерфейс, нижняя навигация на мобильных и PWA-иконки для iOS/macOS.

## Локальный запуск

Проект не требует сборки или зависимостей:

```bash
python3 -m http.server 4173
```

Затем откройте <http://127.0.0.1:4173>.

## Данные

Подключение к общей базе уже задано в `app.js`. Для новой базы выполните [`supabase/schema.sql`](supabase/schema.sql) в Supabase SQL Editor и используйте только publishable/anon key.

- Перевод слова и определения — MyMemory Translation API.
- Английское определение и транскрипция — Dictionary API.
- В таблице `public.cards` используются `id`, `word`, `translation`, `definition`, `phonetic`, `created_at`.
