# Контекст проекта

## Продукт

**English Cards** — личное веб-приложение для изучения английских слов и фраз. Основной путь: добавить слово в разделе «Перевод», при необходимости поправить автоподставленные данные, затем повторять перемешанную колоду в разделе «Учить».

## Технологии

- Статические `index.html`, `styles.css`, `app.js`, без сборщика.
- Supabase REST API — общая база карточек.
- Wiktionary MediaWiki API — первичный источник перевода и английского определения одного значения.
- MyMemory Translation API — русская формулировка определения; fallback для перевода.
- Dictionary API — fallback для определения и транскрипции.
- PWA: `manifest.webmanifest`, иконки, `theme-color: #f5f5f7`.
- GitHub Pages — публикация после push в `main`.

## Интерфейс и визуальная система

- Язык интерфейса — русский; стиль — лёгкий, минималистичный, с ориентиром на Apple Translate.
- Акцентный цвет — `#00B8C8`, фон — `#f5f5f7`, поверхность карточек и полей — белая.
- Разделы в порядке: **«Перевод»**, **«Словарь»**, **«Учить»**. На мобильных это одна нижняя навигация из трёх равных пунктов.
- Верхний логотип — `logo-outline.svg`; справа индикатор подключения к базе: акцентный при подключении, красный при ошибке.
- В «Переводе» нет общей белой панели: форма размещена на сером фоне, каждое поле — отдельная белая поверхность с тонкой обводкой и скруглением `16px`.
- Поля во всём приложении унифицированы: белый фон, `1px` обводка `var(--line)`, `16px` скругление, акцентное состояние фокуса. Типографика значения повторяет карточки: слово — тёмное крупное, перевод — акцентный, описание — серое компактное.
- В «Словаре» есть заголовок «Словарь», карточки, редактирование (карандаш), удаление (крестик) и кнопка «Показать ещё» только когда есть следующая страница.
- Диалог редактирования оформлен как карточка. В нём остаётся только метка «Редактирование карточки», без большого заголовка и пояснения под кнопкой. Кнопки «Добавить» и «Сохранить изменения» — контурные, в стиле «Перемешать».

## Учебный режим

- Карточки проходят одну перемешанную очередь без повторов в текущей сессии.
- Первый тап / `Enter` / `Space` переворачивает карточку; следующий — открывает следующую.
- Показывается прогресс «Показано N из M».
- После последней карточки появляется квадратная финальная карточка «Все слова изучены!» и кнопка «Перемешать» для новой сессии.
- Учебная карточка квадратная (`max-width: 610px`), длинные значения подстраиваются без изменения стандартного размера остальных слов.

## Данные и ограничения

- Карточка содержит `id`, `word`, `translation`, `definition`, `phonetic`, `created_at`. Поле `example` больше не используется интерфейсом и не должно возвращаться в формы.
- Начальная загрузка словаря — `PAGE_SIZE = 60`; кнопка «Показать ещё» подгружает следующую страницу.
- Автоматический поиск запускается при изменении английского слова. Сначала приложение ищет первую пару «значение + русский перевод» в Wiktionary; если формат статьи не подходит, бесшовно использует MyMemory и Dictionary API. При ошибке пользователь может заполнить перевод вручную.
- Общая коллекция намеренно открыта без авторизации для личного приложения. Не менять RLS без отдельного запроса.

## Текущий процесс

Работа идёт локально. Коммит создаётся только после явного подтверждения пользователя; отправка на GitHub требует отдельного явного указания («на GitHub», «push» и т. п.).
